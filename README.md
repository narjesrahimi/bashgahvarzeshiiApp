## Sogand Gym Management System (SGMS)


Welcome to the Sogand Gym Management System. This application is designed to efficiently manage member profiles, sports subscriptions, workout selections, and personalized diet plans.

Features & Usage

# 1. Member Registration

To get started, navigate to the “User” (کاربری) section to register a new member. Please provide the member’s first name, last name, mobile number, and other required details.

# 2. Sport Selection

Once registered, you can assign one or more sports to the member. The system will automatically calculate and store the total cost based on the selected sports.

# 3. Subscription Management

To access club services, members must have an active subscription. You can easily register and track subscription types and their respective expiration dates.

# 4. Diet Plan

After a valid subscription is registered, members can access a personalized diet plan by entering their age, height, and weight.

Note: Access to the diet plan feature is restricted to members with an active subscription only.

# 5. Member Dashboard

Members can log in to their personal dashboard using their mobile number to view their:

Selected sports

Total cost

Subscription type and expiration date

# 6. Edit & Delete

Edit: You can update member information at any time.

Delete: Deleting a member will permanently remove all associated data, including their sports selections and subscription history.

Important Guidelines

Mobile Number: Please ensure the mobile number is entered correctly, as it serves as the primary identifier.

Numeric Fields: Fields such as age, height, and weight must strictly contain numeric values.

Data Accuracy: Please verify all data before finalizing the registration.

Subscription Requirement: A valid subscription is mandatory to unlock the diet plan feature.
